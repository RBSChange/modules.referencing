Import d'un fichier de redirections

Le fichier doit être au format CSV avec les propriétés suivantes :

  • la 1ere colone contient l'ancienne URL et commence par le caractère /
  • la 2ème colone contient l'URL de destination et commence par http://
  • il n'y a pas d'entête de colonne
  • il n'y a pas de délimiteur de champ (pas de double quote autour des valeurs)
  • le séparateur est le point-virgule ";"
  • Exemple de ligne:
    /anciennepage.html;http://mon.site.fr/nouvellepage.html
